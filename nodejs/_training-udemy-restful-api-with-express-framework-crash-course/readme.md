## S2/L6 Install Server Setup

* nodemon
* string interpolation with backticks

in local shell:

```
$ npm install --save-dev nodemon
```

in package.json:

```
...
"scripts" : {
	...
	"start": "nodemon ./index.js"
}
...
```

in local shell:

```
$ npm run start
```

in web browser:

```
http://localhost:3000
```
